Frequently Asked Questions

  • How do I schedule a repeating event with Cyclone?

    1. Open Cyclone by using the Google Calendar Toolbar to click on the Cyclone icon.
    2. Click on the event you'd like to repeat.
    3. A card will appear for you to specify a few options for repeating the event.
    4. Select a schedule to define cycle days, special days, and holidays for the repeating event.

      Events will not repeat on special days or holidays.
    5. Select a pattern for the repeating event.

      Events will not repeat on special days or holidays.
    6. Select the last day the event should repeat.
    7. Click Repeat Event.

      A confirmation message should appear once the recurring events have been added to your Calendar.

      You may have to refresh your web browser to see the newly populated events:

  • How do I delete a Cyclone-created event?

    You can delete a recurring event created by Cyclone through the standard Google Calendar event deletion process.

    1. Click on the event you'd like to delete.
    2. Click on the trashcan icon.
    3. Specify whether you'd like to delete the single event, the event and all following connected events, or all connected events both before and after this event.
    4. Click OK.

      The event(s) will be removed from your calendar.

  • How do I make sure my recurring event moves with a rotating period?

    Some schedules have rotating periods. For instance, your schedule may have a 6th period that meets in the afternoon on Mondays, Tuesdays, and Thursdays, but rotates to meet in the morning on Wednesdays and Fridays to accommodate student early dismissals for Athletics.

    Cyclone will prompt you for your intentions if you attempt to repeat an event that overlaps with either the morning or afternoon ocurrence of such a rotation.

    1. Open Cyclone and click on an event to repeat.

    2. If Cyclone detects that your event overlaps with a rotating period you'll be prompted to clarify how you'd like your event to accommodate this rule.

    3. If your event should ignore the rotating period and repeat at the same time every cycle day, deselect the option to repeat with the period label.

    4. If the meeting time of your repeating event should move with the period label, indicate which portion of the period should be occupied by your event.

    5. Click Repeat Event.
      Your event should populate correctly on according to the specified period rotation pattern.
    6. Cyclone will offer a prompt on either side of the rotating period.

      For instance, in this scenario the selected event overlaps with a time that would typically hold a rotating 6th period.

      Cyclone detects this and offers an option to specify how the period rotation should be handled.

  • Can I schedule a recuring event where I'm Free?

    Cyclone will preserve all event settings for the events that are created for a recurring event. If the event you'd like to repeat is Free, all recurring events will also be Free.

  • How do I generate a printable planner?

    Cyclone Almanac allows you to quickly generate a printable planner with any collection of events from your Google Calendar. This Cyclone feature is coming soon!

  • How does a Workspace Admin configure a recurring schedule?

    After installing Cyclone as a Workspace Add-On, configuring a unique cycle-based schedule is a quick and painless process.

    For organizations with multiple calendars (like schools with multiple divisions), a Google Workspace Admin can repeat this process for each unique cycle schedule.

    1. The Google Workspace Admin for your organization should visit their Cyclone Configuration Web Interface, then select a Google Calendar to define a cycle schedule.

      Cycle days are defined by Google Calendars with All-Day events to match a particular cycle label.

    2. Cyclone will pre-populate calendar start date, end date, cycle labels, and non-cycle labels.
    3. Add/delete cycle labels and non-cycle labels as needed.
    4. Indicate your time zone, and select optional rotating meeting configurations, then click Submit.
    5. Use the color-coded feedback to verify your cycle schedule is correct.
    6. All workspace users can now select this schedule as an option for repeating events.
